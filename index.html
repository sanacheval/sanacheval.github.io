Search Engine in Java

The search engine was implemented by many functions. It first crawls and indexes through all the data provided. Starting at one url, it indexes the words of that url. Using a class MyWebGraph provided by the instructor, it creates a directed graph using adjacency lists of all the urls. Thus after indexing the words of a given url, it moves to the next edge of that vertex, which is not indexed yet, and proceeds to index it. It is a recursive method.

Next, the search engine ranks each page. It does this by considering whether it was cited by many other pages or whether it has links to a large number of other pages within it. A formula provided to us was used to calculate the exact page rank, seeing a page that is cited by many others as better, and one that has a large number of external links in it as worse.

Finally, the search engine computes the results of a word search, taking as input a String and returning an ArrayList of urls that contain the word ordered by the ranking system from before. In order to sort the urls by rank, MergeSort is implemented and it runs in O(n*log(n)) time complexity.

Operating System in C

I took a class focused on Operating Systems. As we learnt how they work, we simultaneously coded one as our class project. I was provided with a basic starter shell on which I built a running operating system in C. It has a scheduling infrastructure that runs in First Come First Serve (FCFS), Shortest Job First with Job Ageing (SJF) and Round Robin (RR). The scheduler was multithreaded. Finally I built a paging infrastructure where I set up a backing store which would then load scripts to the frame memory when they were called to run.

Building an OS helped me understand how far ahead we have come since the first understanding of how it would work. It felt impossible to imagine an OS that ran as smoothly and fast as our current computers have.

In a class focused on ArcGIS Pro and Geospatial Data Analysis, the final project was assessing an area for the building of a Ski Resort.
